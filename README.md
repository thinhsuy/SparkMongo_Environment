# SparkMongo Environment
## Setting up environment
``` python
import os
os.environ['HADOOP_HOME'] = '/path/to/pyspark'
os.environ['JAVA_HOME'] = '/path/to/java'
os.environ['MONGO_HOME'] = '/path/to/mongo'
```
